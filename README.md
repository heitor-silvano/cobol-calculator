## cobol-calculator
### How to run
- On VSCode, install the `bitlang.cobol` extension
- On main folder, execute `cobc -x {FILE_NAME}` to compile file
- On main folder, execute compiled file `./{COMPILED_FILE_NAME}` to execute program
